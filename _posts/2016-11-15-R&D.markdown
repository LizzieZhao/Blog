### 程序题
1. 阅读下面代码，然后写出程序运行结果。
````javascript
function asyncFunc(str1, str2, callback) {
  console.log(str1);
  setTimeout(function(){
    console.log(str2);
    callback();
  },1000);
}

asyncFunc('A','B',function() {
  asyncFunc('C','D',function() {
    console.log('E');
  });
});

asyncFunc('F','G',function() {
  console.log('H');
});
````

**结果：** 'A''F''B''C''G''H''D''E'
